Introduction

Welcome to Programming hub! The main purpose of this website is to assist the students in their journey of Web Development. This website contains resources from a wide range of sources, providing step-by step tutorials to master the skills of Front-end and Back-end Development.

Front-end Development

We will start our journey of Web Development with Front-end Development. Now, Front-end Development is more of a primary approach to the construction of a website, therefore, easier to learn.

Main Topics of Front-end Development:

  • HyperText Markup Language or HTML
  • Cascading Style Sheets or CSS
  • JavaScript or JS
